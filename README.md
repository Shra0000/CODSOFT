# CodSoft Python Projects

## Description
This repository contains multiple beginner-level Python projects developed
as part of the CodSoft internship program. The projects are designed to improve
understanding of Python basics, logic building, and graphical user interface
(GUI) programming.

Each project focuses on solving a real-life problem using simple and clear code.

## Projects Included

### 1. To-Do List Application (GUI)
A simple Python GUI-based To-Do List application using Tkinter.
It allows users to:
- Add tasks
- Delete tasks
- Manage daily activities using a user-friendly interface

### 2. Calculator Application
A basic calculator built using Python that performs:
- Addition
- Subtraction
- Multiplication
- Division

This project helps in understanding arithmetic operations and user input handling.

### 3. Password Generator
A Python-based password generator that creates strong and random passwords.
It helps users generate secure passwords using:
- Letters
- Numbers
- Special characters

## Technologies Used
- Python
- Tkinter (for GUI applications)
- Standard Python libraries

## How to Run the Projects
1. Install Python 3.x
2. Clone or download this repository
3. Open the project folder
4. Run the desired file using:
   python filename.py

Example:
   python todo.py
   python calculator.py
   python password_generator.py
